# Browser History Management using Memento Pattern

## Overview

This project demonstrates the Memento Design Pattern applied to a web browser to manage browsing history. It allows users to navigate back and forth between previously visited webpages by storing and restoring the browser's state.


## Files

- **`Browser.java`**: The originator class that manages the current webpage and can create and restore mementos.
- **`HistoryManager.java`**: The caretaker class that manages the stack of mementos to allow undo functionality.
- **`Main.java`**: Demonstrates the use of the browser and history manager to navigate and manage browser history.

## How to Use

1. Clone the repository to your local machine.
2. Navigate to the project folder.
3. Compile the Java files.
4. Run the application.
